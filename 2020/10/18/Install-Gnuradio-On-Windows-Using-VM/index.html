Use Gnuradio On Windows Using VM

Posted on 2020-10-18 Edited on 2020-10-20 Views:
Symbols count in article: 1.2k Reading time ≈ 1 mins.

Introduction

Previously, people used to install Gnuradio on Linux. One of the reason (or said the only reason) is that there are lots of OTT (out-of tree) library needed to be complied only in the linux environment.
Although there’re indeed some way to use USRP on Windows (for example, we can use USRP as Radio IO on Matlab), to use Gnuradio & USRP, people who used windows before are requested to install a Linux System e.g. ubuntu side by side windows.

To use Gnuradio and USRP without install ubuntu side by side windows so that other works running on windows can still continue to work, I follow the way that installing a Ubuntu VM on windows. We skip the part of installing ubuntu VM and focus on how to make the Ubuntu system communicate with the USRP plugged on Windows.

1, Let the Windows find the USRP first

Supposed we have a USRP N210 (with IP address 192.168.10.16) plugged on a NIC, we first assign an IP address 192.168.10.2 with subnet mask 255.255.255.0 as shown in Fig.1. This step is to make the PC stand with N210 in a same subnet 192.168.10.x. After that, Windows can PING the N210 as shown in Fig.2

2, Configure the VM network

We configure the second NIC of VM as the way in Fig.3. In short, we turn on the NIC bridging.

3, In VM ubuntu, set the NIC with IP 192.168.10.6.

We can successfully PING the N210 Fig.4

4 Use UHD command to find N210

Type uhd_find_devices, we can find the N210 on Ubuntu as shown in Fig.5. Enjoy it !
